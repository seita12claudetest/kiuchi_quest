# UI Reference Plan

木内の大冒険 DELUXE のUI参考元と、実装へ落とし込む画面仕様を整理する。
他作品の画像をコピーせず、構成・情報設計・導線のみを参考にする。

## 参考サイトの使い分け

| サイト | 主な用途 | 確認したい画面 |
|---|---|---|
| Game UI Database | 実ゲームのUI比較 | インベントリ、ショップ、ステータス、戦闘、スキルツリー |
| Interface In Game | HUDや画面遷移の比較 | マップHUD、ミニマップ、会話、クエストログ |
| UIX HUB | 洗練されたゲームUI比較 | モバイル/PC向けの画面単位UI |
| Flomob | スマホゲームUX | 画面遷移、報酬導線、チュートリアル |
| Dribbble | 見た目の方向性 | ボタン、カード、配色、装飾 |
| Behance | 完成度の高い世界観設計 | 配色、エフェクト、画面モック |
| Pinterest | ムードボード | Pixel UI、Fantasy UI、RPG Menu |
| Figma Community | UIキット調査 | ボタン、メニュー、ステータスカード |

## ビジュアルコンセプト

**ドット絵RPG × 夜の横丁 × 健康管理アプリ × 会社員コメディ**

- 基本背景は暗い紺・黒・茶系。
- 重要UIは金色・琥珀色の装飾枠でRPG感を出す。
- 病院は白・青・薄緑で清潔感を出す。
- カジノは黒・赤・金で派手にする。
- 自宅は暖色で安心感を出す。
- 食料・アイテムはドット絵アイコンで収集感を出す。

### インベントリ

- 左: カテゴリタブ。食料、飲料、薬、装備、素材、キー、魚。
- 中央: ドット絵アイコン付き一覧。
- 右: 詳細パネル。効果、健康値変動、価格、説明。
- 食料はカロリーと栄養を表示する。

### 食事ログ

- 日付・時間帯ごとに食べたものを表示。
- 摂取カロリー、塩分、脂質、糖質を簡易ゲージで表示。
- 健康診断結果と連動して警告を出す。

### ショップ / 居酒屋

- メニュー表風UI。
- 食料アイコン、価格、HP回復、健康影響を並べる。
- 悪化する健康値は赤、改善する健康値は緑で表示する。

### 病院診断

- 健康診断表風レイアウト。
- 血圧、尿酸値、コレステロール、肝機能、血糖値をゲージ化。
- `A` から `E` の総合評価を大きく表示する。
- 危険値は赤い警告トーストを出す。

### 自宅

- 休む、料理、セーブ、食事ログ、図鑑確認のメニュー。
- 暖色背景と落ち着いたUI。
- 料理時は素材アイコンを並べる。

### カジノ

- 黒・赤・金の派手なパネル。
- スロット、ブラックジャック、ポーカー、ルーレットのゲーム選択。
- コイン残高と景品交換所を常時表示する。

### 釣り

- タイミングゲージ、魚影、引きの強さを表示。
- 成功時は魚ドット絵、サイズ、レア度、売値を表示する。

### スキルツリー

- 肉体派、社交派、節制派の3列。
- 取得済みは金、取得可能は青、未解放は灰色。
- SP残量を常時表示する。

### クエストログ

- メイン、サブ、隠しのタブ分け。
- 目的、進捗、報酬、関連NPC、目的地を表示する。
