# ゲーム全体仕様

状態: Approved 1 / 対象: Godot 4再構築

## プレイヤーファンタジー

プレイヤーは木内一として、日本企業と社会の変化を生きる。仕事で能力を磨き、街を歩いて人や店と出会い、心象世界に現れる問題と仲間4人で戦い、新卒から副部長を目指す。

木内は温厚、真面目、少し不器用という芯を持つ。仕事観、出世欲、人付き合い、健康への態度はプレイヤーの行動で変化する。全体の調子は可笑しさ7、社会の厳しさ3とし、過労、病気、災害、失業を軽薄に扱わない。

## 対象プラットフォーム

- 第一対象: Web版（GitHub Pages）
- 第二対象: macOS / Windows
- 入力: キーボード、ゲームパッド、Web版タッチ操作
- エンジン: Godot 4系
- 表現: 16/32bit時代を想起させる2Dピクセルアート。内部基準48×48px、整数倍表示。

## 想定プレイ時間と結末

- 初回クリア: 25〜35時間
- 1988年から現代までの全日を再現せず、重要な週、月次、年度、社会的転機を選んで遊ぶ。
- 副部長昇進だけを正解としない。副部長、現場専門家、仲間に慕われる管理職、健康と家族、独立・転職、後悔を抱えて残留する結末を持つ。

## 主要ループ

1. 朝、自宅または会社でその日の方針を決める。
2. 実際のマップを歩いて目的地へ行く。
3. 仕事、会話、買い物、探索、訓練、戦闘から行動する。
4. 能力、健康、所持金、好感度、時間、クエストが変化する。
5. 退勤後の自由行動と帰宅を行う。
6. 一日の結果、翌日の予定、社会・会社イベントを処理する。
7. 月次評価、年度評価、昇進または異動へつなげる。

## 時間

時間帯は朝、午前、昼休み、午後、夕方、夜、深夜。歩行、方向転換、メニュー閲覧では進まない。主要行動は15、30、60、120分のいずれかを消費する。営業時間、出勤義務、イベント発生、敵出現は時刻と曜日を見る。

平日は午前・午後に勤務義務を持つ。無断で会社を離れれば自由には動けるが、信用、評価、給与、イベントへ影響する。土日祝、休暇、出張、病欠は別スケジュールとする。

## 育成値

- 仕事: 評価、信頼、専門性、社内調整
- 人間: 体力、ストレス、健康、所持金
- RPG: レベル、HP、集中力、行動速度、各適性
- 関係: NPC別好感度、信頼段階、同行可否
- キャリア: 役職、所属、担当、月次実績、勤続年数

## 能力適性

- 論理: システム、数値、分析
- 対人: 営業、顧客、説得
- 根性: 疲労、圧力、継続
- 政治: 稟議、根回し、社内力学
- 健康: 酒、病気、長期戦
- 時代: 技術・社会変化への適応

## 収集

- 人物録: 出会った人物、関係、印象的な会話
- 問題図鑑: 遭遇した心象モンスター、弱点、解決法
- 食事録: 店、料理、年代、健康効果
- 技術史: AS/400、PC、ネット、携帯、クラウド、生成AI
- 会社史: 辞令、異動、成功、失敗、仲間
- 街の記憶: 店舗、看板、再開発、閉店、開店

収集率だけを報酬にせず、新会話、戦術、店の商品、仕事術、回想を解放する。

## 役職

新卒社員 → 平社員 → 主任 → 係長 → 課長代理 → 課長 → 副部長。

昇進は単一経験値では決めない。最低勤続期間、実績、専門性、上司・同僚の信頼、健康、重要イベントの結果を見る。昇進しない人生、異動、専門職ルートも将来拡張可能にする。

## セーブ

- 自宅の就寝時に自動セーブ
- マップ切替と戦闘開始前に復旧ポイント
- 手動セーブ3枠
- WebはIndexedDB、デスクトップはユーザーデータ領域
- 仕様バージョン付きマイグレーション
