# 木内の大冒険 DELUXE

**昭和横丁 × 健康管理 × 会社員RPG** をテーマにした、ブラウザだけで遊べる1画面RPGです。

矢印キーまたは画面上の方向ボタンで横丁を探索し、飲み会の誘惑と戦いながら、血圧・尿酸・コレステロールを管理します。

## 起動方法

`index.html` をブラウザで開くだけで遊べます。ビルドや外部ライブラリは不要です。

ローカルサーバーを使う場合は、リポジトリのルートで以下を実行します。

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## 実装済み

- マップ探索と時間・曜日進行
- 金曜日に発生しやすい飲み会バトル
- レベル、経験値、HP、攻撃力、防御力、所持金
- 血圧、尿酸、コレステロール、腹囲係数
- 敵ごとの強さ、報酬、健康への悪影響、ランダムドロップ
- アイテム、ショップ、アイテム図鑑、敵図鑑
- 5種類のクエストと報酬
- 病院イベントと高血圧時の強制搬送
- ジム・深呼吸ミニゲーム
- NPC好感度
- 称号、実績、隠しボス
- 3枠セーブ、AUTOセーブ、旧プロトタイプのセーブ互換
- 複数エンディング
  - 健康優良社員エンド
  - 飲み会キングエンド
  - 生活改善エンド
  - 再検査エンド
  - 真エンド
- キーボード・スマートフォン操作
- 簡易サウンドエフェクト
