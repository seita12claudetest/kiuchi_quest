# Noto Sans JP

- Source: https://github.com/google/fonts/tree/main/ofl/notosansjp
- Creator: Adobe / Google Fonts (Noto Project)
- Retrieved: 2026-07-20
- File: `NotoSansJP-Variable.ttf` (variable font, weight axis)
- License: `OFL.txt` (SIL Open Font License 1.1)
- Intended project use: default UI font for the Godot project, so Japanese text (dialogue, HUD, menus) renders correctly in the Web (HTML5) export, which has no OS font fallback unlike desktop builds.

## License summary

The SIL Open Font License permits embedding, bundling, and redistributing the font as part of this game, including in commercial products, without royalty, as long as the font itself is not sold on its own and `OFL.txt` is kept alongside it. This summary is retained for project auditing; `OFL.txt` is the authoritative license text.
